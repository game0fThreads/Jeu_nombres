Le Numéro - Les règles
Le Numéro est un jeu où vous devez utiliser votre logique pour deviner un code secret à 4 chiffres choisi par l’ordinateur. Le code secret est formé avec des chiffres de 1 à 9 et chaque chiffre apparaît au plus une fois.

Vous devez deviner ce code secret à travers plusieurs tentatives. Une tentative consiste à proposé un nombre (le code secret à 4 chiffres). L’ordinateur vous indique dans sa réponse, combien de chiffres vous avez deviné sur la même position et combien de chiffres vous avez deviné sur une position différente.

Utilisez les réponses de l’ordinateur pour deviner le code secret avec moins de tentatives possible. Bonne chance!
